# Dylan Games

Quiet browser-games hub for games.dylanwlim.com.

Dylan Games is a focused browser-games hub with a playable Snake starter and a registry for future lightweight games. The hub is separate from dylanwlim.com and remains in soft launch until the main site intentionally links to it.

## Start here

- Website: [https://games.dylanwlim.com](https://games.dylanwlim.com)
- Discussions: [ask questions and suggest guide improvements](https://github.com/dylanwlim/dylan-games-docs/discussions)
