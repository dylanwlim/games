# Access And Updates

## Public Entry Points

- Public site: [https://games.dylanwlim.com](https://games.dylanwlim.com)
- Public discussions: [https://github.com/dylanwlim/games-docs/discussions](https://github.com/dylanwlim/games-docs/discussions)

## Current Use

1. Open the public hub and choose Play Snake.
2. Use arrow keys or the on-screen controls to move.
3. Use New Game to restart, then browse All Games, Discover, or a genre page.

## How These Docs Stay Current

- Source guide files live under `docs-public/`.
- Public docs publishing is manual while automatic publishing is paused.
- Each refresh should capture a current homepage screenshot before publishing.
- Publishing uses only the approved guide files and homepage screenshot asset.
